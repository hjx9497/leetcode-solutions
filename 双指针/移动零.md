## 移动零 (LeetCode 283)

### 题目描述
给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。必须**原地操作**，不能拷贝数组。

**示例：**
```
输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
```

### 解法一：双指针（覆盖法）

```java
public void moveZeroes(int[] nums) {
    int j = 0; // 指向非零元素要填充的位置
    for (int i = 0; i < nums.length; i++) {
        if (nums[i] != 0) {
            nums[j] = nums[i];
            j++;
        }
    }
    // 剩余位置补零
    for (int i = j; i < nums.length; i++) {
        nums[i] = 0;
    }
}
```

### 解法二：双指针（交换法，更简洁）

```java
public void moveZeroes(int[] nums) {
    int j = 0;
    for (int i = 0; i < nums.length; i++) {
        if (nums[i] != 0) {
            // 交换 nums[i] 和 nums[j]
            int temp = nums[i];
            nums[i] = nums[j];
            nums[j] = temp;
            j++;
        }
    }
}
```

### 复杂度分析
- 时间复杂度：O(n)，一次遍历
- 空间复杂度：O(1)，原地操作

### 关键点
- 保持非零元素的相对顺序
- 必须原地修改，不能使用额外数组
- 交换法可以避免最后单独补零的循环
