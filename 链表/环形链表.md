# 环形链表 (LeetCode 141) 

## 题目描述
给你一个链表的头节点 `head`，判断链表中是否有环。

如果链表中有某个节点，可以通过连续跟踪 `next` 指针再次到达，则链表中存在环。返回 `true` 表示有环，`false` 表示无环。

## 解题思路（哈希表法）

### 核心思想
使用哈希表（`HashSet`）记录已经访问过的节点。遍历链表：
- 如果当前节点已经在哈希表中，说明之前访问过这个节点 → **有环**，返回 `true`。
- 如果当前节点不在哈希表中，将其加入哈希表，继续遍历下一个节点。
- 如果遍历到 `null`（链表结束），说明没有环，返回 `false`。

### 为什么可以用哈希表？
环的本质是某个节点被重复访问。哈希表可以 O(1) 的时间判断一个节点是否已经出现过。

### 手动模拟
输入链表：`3 → 2 → 0 → -4`，且 `-4` 指向 `2`（形成环）。

| 步骤 | 当前节点 | visited 集合 | 是否重复？ | 操作 |
|------|----------|--------------|------------|------|
| 1 | 3 | {} | 否 | 加入 {3}，移动到 2 |
| 2 | 2 | {3} | 否 | 加入 {3,2}，移动到 0 |
| 3 | 0 | {3,2} | 否 | 加入 {3,2,0}，移动到 -4 |
| 4 | -4 | {3,2,0} | 否 | 加入 {3,2,0,-4}，移动到 2 |
| 5 | 2 | {3,2,0,-4} | **是** | 返回 `true` |

## 复杂度分析
- **时间复杂度：O(n)**  
  最坏情况下需要遍历所有节点（无环）或直到重复节点（有环）。
- **空间复杂度：O(n)**  
  最坏情况下需要存储所有节点（无环时）。

> **对比快慢指针：** 哈希表法更直观易懂，但需要 O(n) 额外空间；快慢指针只需要 O(1) 空间（见下文）。

## 代码实现

```java
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
import java.util.HashSet;
import java.util.Set;

public class Solution {
    public boolean hasCycle(ListNode head) {
        Set<ListNode> visited = new HashSet<>();
        ListNode current = head;
        
        while (current != null) {
            // 如果当前节点已经访问过，说明有环
            if (visited.contains(current)) {
                return true;
            }
            // 将当前节点加入哈希表
            visited.add(current);
            // 移动到下一个节点
            current = current.next;
        }
        // 遍历结束，没有重复节点，说明无环
        return false;
    }
}
``` 

## 关键点 / 易错点
- **存储的是节点对象，不是节点值**：因为不同节点的值可能相同（如 `[1,2,1]` 无环但值重复），必须用节点本身作为判断依据。
- **`HashSet<ListNode>`**：Java 中 `HashSet` 基于 `hashCode()` 和 `equals()`，`ListNode` 默认继承 `Object` 的实现，比较的是对象引用（内存地址），这正是我们需要的。
- **空链表判断**：`while` 循环条件 `current != null` 已经处理了空链表的情况，返回 `false`。

## 扩展：快慢指针解法（空间 O(1)）

```java
public boolean hasCycle(ListNode head) {
    if (head == null || head.next == null) return false;
    ListNode slow = head;
    ListNode fast = head.next;
    while (slow != fast) {
        if (fast == null || fast.next == null) return false;
        slow = slow.next;
        fast = fast.next.next;
    }
    return true;
}
```

或者更简洁的写法（起始相同）：
```java
public boolean hasCycle(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) return true;
    }
    return false;
}
```
