## 找到所有数组中消失的数字 (LeetCode 448)

### 题目描述
给你一个含 `n` 个整数的数组 `nums`，其中 `nums[i]` 在区间 `[1, n]` 内。请你找出所有在 `[1, n]` 范围内但没有出现在 `nums` 中的数字，并以数组的形式返回。

**示例：**
```
输入：nums = [4,3,2,7,8,2,3,1]
输出：[5,6]
```

**说明：** 要求不使用额外空间且时间复杂度为 O(n)。

---

### 解题思路

#### 方法：原地修改（利用索引标记）
由于数组元素在 `[1, n]` 范围内，我们可以利用**索引**来标记某个数字是否出现过。

**核心思想：**  
遍历数组，对于每个元素 `nums[i]`，将其对应的索引位置（`|nums[i]| - 1`）上的数字变为负数。这样，最终仍然为正数的索引位置 `i+1` 就是缺失的数字。

**步骤：**
1. 遍历数组，对于每个 `num = Math.abs(nums[i])`：
   - 计算索引 `index = num - 1`。
   - 如果 `nums[index] > 0`，将其取负（标记为已出现）。
2. 再次遍历数组，找出所有仍为正数的元素，其索引 `i+1` 即为缺失的数字。

**注意：** 需要取绝对值是因为之前的操作可能已经把某些数字变成了负数。

---

### 复杂度分析
- **时间复杂度：** O(n)，两次遍历。
- **空间复杂度：** O(1)，除返回数组外不使用额外空间。

---

### 代码实现

```java
import java.util.ArrayList;
import java.util.List;

public List<Integer> findDisappearedNumbers(int[] nums) {
     List<Integer> list = new ArrayList<>();
     for(int i = 0; i < nums.length ; i++>){
        int index = Math.abs(nums[i]) - 1;
        if(nums[index] > 0){
            nums[index] = -nums[index];
        }
     }
     for(int i = 0; i < nums.length; i++){
        if(nums[i] > 0 ){
            list.add(i + 1);
        }
     }

     return list;

}
```

### 关键点/易错点
- 使用 `Math.abs()` 避免索引越界（因为之前可能已将某些数字变为负数）。
- 只标记未标记过的位置（`nums[index] > 0`），避免重复标记导致正负混乱。
- 返回的列表顺序无关紧要。
